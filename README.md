## 💻 Requirements

Before using the bot be sure you have the following:
* Create a bot on Discord Developer Portal and save the Token.
* Have a Discord server and save its guild id (Right Click + Copy ID on server´s logo).
<br>

## 🚀 Setup

Steps to setup:

1. Clone the repository.
2. Execute `pip3 install -r requirements.txt` in the project folder.
3. Replace `<guild_id>` and `<bot_token>` in config.ini with your values.
4. Run bot with `python3 pycord.py`.
<br>

## ☕ Use of the bot

With the bot running on the server you can:

* Search any anime available in the server with `/search <query>`on discord´s chat.
* See full information and data of an anime.
* Request an episode link from a button and see with your friend on [OpentogetherTube](https://opentogethertube.com) instantly.
